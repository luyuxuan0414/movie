# movie
威秀影城官網暨訂票系統

此影城網站結合訂票系統，提供迅速的網路會員註冊、登入、資料查詢，以及忘記密碼功能。
影迷可輕鬆查閱影城介紹和電影資訊，並透過會員訂票系統預約座位。
最後，訂單紀錄系統可以方便會員查詢及修改訂票歷史。
